# Taikai

Taikai is a wrapper around the awesome ArchUnit and provides a set of common rules for different technologies.

Maven Usage
-------------------

```xml
<dependency>
  <groupId>com.enofex</groupId>
  <artifactId>taikai</artifactId>
  <version>0.0.1</version>
</dependency>
```
